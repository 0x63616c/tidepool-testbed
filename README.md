# tidepool-testbed

The safe sandbox repo that [Tidepool](https://github.com/0x63616c/tidepool)'s coding agents practice on. It is a tiny pure-function TypeScript utility library with vitest, carrying the **same mechanical rails** as the main repo (Biome, strict TypeScript, vitest, commitlint, CI-gated PRs, branch protection) so agent work here is graded identically — but objective pass/fail and harmless on failure. It owns no infrastructure and exists purely as an endless supply of easy, well-specified tickets plus a permanent safe integration-test target.
